# Desert Safari Adventure Frontend

This is the frontend code for [dsrtsafariadventure.com](https://dsrtsafariadventure.com).  
It includes all HTML, CSS, JavaScript, and image assets for the website.

## Description

- **index.html** – Main homepage.
- **assets/css/style.css** – Main stylesheet.
- **assets/js/** – JavaScript files for site functionality.
- **assets/images/** – All images used across the site.
- **evening-desert-safari/index.html** – Evening desert safari page.
- **vip-red-dune-safari/index.html** – VIP Red Dune Safari page.

## Usage

- Open `index.html` in a browser to view the homepage.
- Ensure the `assets/` folder remains in the same relative location for styles, scripts, and images to load correctly.
